# Hello-scala

## Install on Mac OS X

`$ brew install scala`

## Compile

`$ scala <target file>.scala`

or

`$ fsc <target file>.scala`

If you want to stop fsc deamon, should do following command.

`$ fsc -shutdown`

## Editor

### Emacs

`emacs cask` and `pallet`

`scala-mode2`
